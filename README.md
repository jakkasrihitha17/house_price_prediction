# 🏠 House Price Prediction

My first Machine Learning project built during B.Tech 3rd year (CSE AI & ML)

## 📌 What This Project Does
Predicts house prices based on size, bedrooms, age, and distance using Linear Regression.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📊 Results
- Trained on 500 data points
- R2 Score: ~0.95 (95% accuracy)

## 🚀 How to Run
1. Clone this repository
2. Install libraries: pip install pandas numpy scikit-learn matplotlib
3. Run: python house_price.py
